Route

new Route(path, queryopt)

A Route() represents a particular view in your app, and is a combination of a path (potentially with params) and an optional query.
Parameters
path:string
the path that represents the view you want to show, ex. /profiles/123
query:Object (optional)
key/value pairs representing a URL query string
