Router

Router

new Router(target)

Router() is a singleton class that lets you load content based on the current HREF. It allows you simulate a multipage app from a single page.
Parameters
target:HTMLElement
the element where the desired content will be loaded for all routes

Methods

addRoute(pattern, initializer)

Adds a route handler for the router, which loads a given element when the HREF matches a given pattern. Accepts a pattern to match and a function that will be provided with the queries and params that must return the desired content for the route.
Parameters
pattern:string
string to match route against. string can use params like ":param" in place of route segments. must start with a forward slash. ex: /route/with/:param
initializer:RouteInitializer
function that should create/return the DOM content to be loaded for this route
